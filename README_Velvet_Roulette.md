# Velvet Roulette

## Overview

Velvet Roulette is a premium European (single-zero) roulette game for
entertainment only. It uses virtual Gold and Diamonds and does not
support real-money gambling or cash prizes.

## Features

-   European Roulette (0--36)
-   Realistic wheel and ball physics
-   Google Sign-In & Guest Mode
-   Firebase cloud save
-   Daily Spin & Daily Perks
-   Player levels
-   Gold & Diamonds
-   Google Play Billing
-   Restore Purchases

## Installation

### Android

1.  npm install
2.  npx cap sync android
3.  Open Android Studio
4.  Build APK/AAB

### Web

Upload the Web Package to Vercel, Firebase Hosting, or any static host.

## How to Play

1.  Login with Google or continue as Guest.
2.  Collect Daily Spin and Daily Perks.
3.  Select a chip.
4.  Place one or more bets.
5.  Press Spin.
6.  Win Gold based on European Roulette payouts.
7.  Level up by increasing your total winnings.

## Bet Types

-   Straight
-   Split
-   Street
-   Corner
-   Six Line
-   Dozens
-   Columns
-   Red/Black
-   Even/Odd
-   High/Low

## Currency

Gold is used for betting. Diamonds are premium currency used to purchase
Gold.

## Purchases

All Android purchases are verified securely through Firebase before
rewards are granted. Refunded or cancelled purchases cannot be restored.

## Saving

Google accounts sync progress through Firebase. Guest progress is stored
locally.

## Troubleshooting

-   Login: Check internet and Google Play Services.
-   Purchases: Use Restore Purchases after reinstalling.
-   Sync: Use the same Google account.

## Technologies

HTML, CSS, JavaScript, Capacitor, Firebase, Android Studio, Google Play
Billing, AdMob.
